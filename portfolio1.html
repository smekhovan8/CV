Регистрация облаков точек

ООО "Абаджи Роботик Системс", декабрь 2018

    Регистрация облаков точек (point cloud registration, point cloud matching) — это процесс вычисления такого пространственного преобразования T, которое позволяет оптимальным образом совместить (align) исходное (source) облако точек с целевым (target) облаком точек. Под оптимальным здесь имеется ввиду совмещение, максимизирующее количество соответствующих пар точек (correspondences) в исходном и целевом облаках. Две точки Ps и Pt называются соответствующими, если они являются взаимно ближайшими друг к другу (reciprocal corresponding points) и при этом расстояние между ними не больше некоторого наперёд заданного epsilon.

    Облака точек были получены с помощью датчика профилей, закреплённого на фланце промышленного манипулятора. В качестве сканируемых изделий выступали различные металлические конструкции (строительные балки, кронштейны и т.д.).  В связи с особенностями решаемой задачи время на совмещение облаков было ограничено не очень жёстко, однако требовалась высокая надёжность и точность работы алгоритма. Были испробованы уже реализованные методы из различных библиотек, таких как PCL, Super4PCS и коммерческой библиотеки машинного зрения Halcon. Эксперименты показали, что при произвольном начальном приближении исходного облака ни один из методов при фиксированных параметрах не обеспечивает требуемой надёжности совмещения на всём многообразии сканируемых изделий. Однако на различных данных результаты работы методов часто не коррелировали между собой. То есть на каждом наборе данных хотя бы один из методов давал приемлемый результат. Поэтому мной было решено использовать все эти методы совместно, вызывая их поочерёдно и оценивая качество полученного совмещения (score) до получения желаемого значения. При этом параметры каждого из методов были подобраны так, чтобы давать хороший результат на как можно большем количестве тестовых данных.

    На картинках ниже можно видеть результат работы алгоритма. Синие точки обозначают исходное облако точек, а красные — целевое облако точек. Также на видео (2:36) можно увидеть практическое применение данного алгоритма.

Инструменты разработки: C++, PCL, Halcon, Super4PCS, CLion.

Определение параметров разделки сварного шва

ООО "Абаджи Роботик Системс", июнь 2018

    Разделка кромок — это процесс придания кромкам заготовок, подлежащих сварке, формы, необходимой для полного провара заготовок по сечению. Это важное условие для получения качественного и прочного сварного соединения. Геометрию разделки кромок определяют так называемые зазор и притупление. На рисунке ниже схематично объясняется смысл этих параметров.

    При роботизированной сварке заготовок с заранее подготовленной разделкой кромок появляется необходимость автоматического определения параметров этой разделки. От этого зависит выбор режима сварки, что напрямую влияет на качество будущего соединения. Таким образом, целью этой работы было создание и реализация алгоритма для автоматического определения значений искомых параметров.

    Сварка на роботизированных комплексах осуществляется с применением датчика профилей, закреплённого на подвижном манипуляторе. Про этот вид датчика подробно было рассказано ранее. На выходе он даёт плоские наборы точек (профили) в собственной системе координат. Зная пространственное положение сканера в каждый момент времени, эти профили можно преобразовать в мировую систему координат. Объединение множества таких профилей даёт трёхмерное облако точек, удобное для решения задачи определения искомых параметров. Алгоритм, разработанный мной, основан на аппроксимации облака точек плоскостями и нахождении точек на скосе разделки. Искомые параметры вычисляются в отдельности для каждого профиля, после чего их можно отфильтровать для исключения выбросов (например, с применением медианной фильтрации) и найти среднее значение по всем профилям.

Инструменты разработки: Python 3, NumPy, CLion.

Детекция сварных точек

ООО "Абаджи Роботик Системс", апрель 2018

    Роботизированная сварка является одним из наиболее распространённых применений промышленных роботов. Для обеспечения наибольшей гибкости этого процесса роботов часто оснащают датчиками, которые позволяют качественно заваривать швы, даже если их положение и геометрия немного отличаются от заданной. Существует несколько разновидностей таких датчиков. Из них наиболее полную информацию о пространственном положении и геометрии шва позволяет получить датчик профилей. Подробно о принципе работы этого типа датчика было рассказано ранее. Также ранее было рассказано про калибровку этого датчика, которая необходима для его применения на сварочном роботе. Достаточно знать, что на выходе датчик даёт набор точек (профиль), полученных сечением поверхности плоскостью лазера. Целью этой работы было создание и реализация алгоритмов для нахождения точек сварки по набору таких профилей на соединениях различных типов.

    В основе всех разработанных мной алгоритмов лежит кусочная аппроксимация профилей кривыми первого или второго порядка. После аппроксимации на основе знаний о взаимном расположении кривых определяется несколько возможных кандидатов на роль точек сварки. Окончательное решение о выборе точек принимается исходя из максимизации некоторой интегральной оценки.

    Для быстрого прототипирования и отладки алгоритмов использовалась среда MATLAB, после чего они были перенесены на C++. На картинках ниже можно видеть результаты работы алгоритмов для соединения типа кольцо-арматура и для таврового соединения двух пластин. Также на видео можно видеть демонстрацию алгоритма поиска точек сварного шва для обварки арматурного каркаса.

Инструменты разработки: С++, MATLAB, CLion.

Внешняя калибровка датчика профилей

ООО "Абаджи Роботик Системс", декабрь 2017

    Про принцип работы и назначение датчика профилей (лазерного триангуляционного 2D сканера) подробно было рассказано ранее. Для того, чтобы максимально использовать все возможности такого сканера, его часто крепят на фланец промышленного робота. Это позволяет, при условии синхронизации движения робота и захвата профилей со сканера, использовать 2D сканер в качестве 3D датчика. Однако для этого необходимо откалибровать сканер на роботе, то есть найти преобразование из системы координат сканера в систему координат фланца. Эта процедура называется внешней калибровкой (extrinsic calibration). Целью этой работы было создание алгоритма, который бы позволил проводить такую калибровку в автоматическом режиме.

    Существуют различные способы выполнения внешней калибровки. Часто они, так или иначе, используют некоторый калибровочный объект с известной геометрией. В данной работе использовалась сфера известного радиуса (бильярдный шар).

    Суть реализованного алгоритма заключается в автоматическом сборе профилей со сферы при различных положениях фланца робота и подборе таких параметров преобразования координат, которые бы минимизировали среднеквадратичное отклонение полученного облака точек от поверхности сферы. Мной была разработана двухэтапная процедура поиска целевого преобразования. На первом этапе происходит сбор профилей при фиксированной ориентации сканера. Это позволяет вычислить три угла (roll, pitch, yaw), определяющие матрицу поворота сканера. На втором этапе происходит сбор профилей с различными ориентациями сканера. Это позволяет, используя углы, найденные на предыдущем этапе, вычислить смещение (x, y, z) сканера.

    На анимированных изображениях ниже можно видеть процесс сбора профилей. В качестве датчика профилей использовался промышленный сканер Wenglor MLSL 123. Также на анимациях показан численный поиск решения, которое даёт наименьшее возможное отклонение собранных профилей от поверхности сферы.

Инструменты разработки: С++, MATLAB, CLion.

Применение свёрточной сети для сегментации изображений преформ

ООО "Эйдос-Робототехника", июль 2017

    Целью этой работы было создание системы автоматического контроля качества бутылочных ПЭТ преформ, движущихся по конвейерной ленте. По требованию заказчика контроль должен был производиться по изображениям, получаемым с одной цветной камеры, закреплённой над конвейерной лентой. Преформы при этом движутся по ленте неупорядоченно. Контролируемыми параметрами являются цвет преформы и наличие вкраплений.

    Так как это достаточно комплексная задача, на первом этапе было решено разработать алгоритм отделения изображений преформ от фона, то есть сегментации.

    Преформы могут быть множества различных оттенков, в том числе прозрачные. Конвейерная лента имеет неоднородную текстуру с тёмными и светлыми участками. Также на изображении преформы могут возникать блики различной формы и площади. Все эти факторы сильно усложняют задачу сегментации. В таких условиях использование простых методов типа сегментации по яркости или цвету даёт неприемлемый результат. В связи с этим было принято решение использовать для сегментации свёрточную нейронную сеть.

    За основу была взята работа, в которой авторы использовали свёрточную нейросеть для сегментации изображений лиц. Их модель, в свою очередь, была основана на архитектуре SegNet. Для реализации модели был использован фреймворк TensorFlow.

    Мной была произведена адаптация модели с целью уменьшения числа варьируемых параметров и, соответственно, сокращения требуемого объёма памяти и времени обучения.

    Для обучения модели была сформирована и вручную размечена выборка из порядка 100 изображений преформ. Выборка была искусственно аугментирована при помощи операций сдвига, отражения и размытия. Все изображения были разделены на обучающую и тестовую выборки в соотношении 0.7/0.3. Результаты сегментации для тестового набора данных показаны на рисунках ниже. Как видно, обученная модель даёт неплохой результат даже для прозрачных преформ.

Инструменты разработки: Python 3, TensorFlow, Eclipse.
